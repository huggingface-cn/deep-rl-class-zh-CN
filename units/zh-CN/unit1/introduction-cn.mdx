# 深度强化学习简介 [[introduction]]

<img src="https://huggingface.co/datasets/huggingface-deep-rl-course/course-images/resolve/main/en/unit1/thumbnail.jpg" alt="Unit 1 thumbnail" width="100%">


欢迎来到人工智能中最引人入胜的话题：**深度强化学习。**

深度强化学习是一种机器学习，其中智能体通过在环境中 **做出动作** 和 **搜寻结果** 来学习 **怎样行动** 。

在第一单元中，**你将学习深度强化学习的基础知识。**


然后，你将使用强化学习库 <a href="https://stable-baselines3.readthedocs.io/en/master/"> Stable-Baselines3 </a> **训练你的深度强化学习智能体，让月球着陆器正确降落在月球上**。


<img src="https://huggingface.co/datasets/huggingface-deep-rl-course/course-images/resolve/main/en/unit1/lunarLander.gif" alt="LunarLander">

最后，你将 **将这个训练好的智能体上传到 Hugging Face Hub 🤗，这是一个免费的开放平台，人们可以在其中共享 ML 模型、数据集和示例。**

在深入执行深度强化学习智能体之前，**掌握这些要素**至关重要。本章的目标是为你打下坚实的基础。


完成本单元后，在奖励单元中，你将**能够训练 Huggy 狗🐶 叼棍子和他一起玩🤗**。
