# 奖励内容：学习使用 Unity 和 MLAgents 创建你自己的环境

**你可以使用 Unity 和 MLAgents 创建自己的强化学习环境**。但是，使用 Unity 等游戏引擎一开始可能会让人望而生畏，但你可以按照以下步骤顺利学习

## 第 1 步：了解如何使用 Unity

- 学习 Unity 的最佳方式是学习 ["Create with Code" 课程](https://learn.unity.com/course/create-with-code)： 这是一系列面向初学者的视频，**你将在其中使用 Unity 创建 5 个小游戏**。

## 第 2 步：使用本教程创建最简单的环境

- 然后，当你知道如何使用 Unity 时，你可以使用本教程创建你的[第一个基本 RL 环境](https://github.com/Unity-Technologies/ml-agents/blob/release_20_docs/docs/Learning-Environment-Create-New.md)。

## 第 3 步：迭代并创建良好的环境

- 现在你已经创建了第一个简单的环境，你可以使用 [MLAgents 文档(尤其是设计智能体和智能体部分)](https://github.com/Unity-Technologies/ml-agents/blob/release_20_docs/docs/)在更复杂的环境中进行迭代
- 此外，你可以学习 [Adam Kelly](https://twitter.com/aktwelve) 的免费课程[“创建一个蜂鸟环境”](https://learn.unity.com/course/ml-agents-hummingbirds) 


玩得开心！如果你创建自定义环境，请毫不犹豫地将它们分享到 discord 的 `#rl-i-made-this`  频道。
