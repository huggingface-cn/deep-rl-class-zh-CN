# 尝试有趣环境

我们在这里提供了一些有趣的环境，你可以尝试在这些环境中训练你的智能体：

## MineRL

<img src="https://huggingface.co/datasets/huggingface-deep-rl-course/course-images/resolve/main/en/unit12/minerl.jpg" alt="MineRL"/>


MineRL 是一个 Python 库，提供了与视频游戏 Minecraft 进行交互的 Gym 接口，并附带了人类游戏数据集。
每年都会有使用这个库的挑战。请查看[网站](https://minerl.io/)

要开始使用这个环境，请查看以下资源：
- [什么是 MineRL？](https://www.youtube.com/watch?v=z6PTrGifupU)
- [MineRL 的第一步](https://www.youtube.com/watch?v=8yIrWcyWGek)
- [MineRL 文档和教程](https://minerl.readthedocs.io/en/latest/)

## DonkeyCar 模拟器


<img src="https://huggingface.co/datasets/huggingface-deep-rl-course/course-images/resolve/main/en/unit12/donkeycar.jpg" alt="Donkey Car"/>
Donkey 是一个用于遥控汽车爱好者的自动驾驶汽车平台。
该模拟器版本是基于 Unity 游戏平台构建的。他使用 Unity 内部的物理和图形，并连接到 Donkey Python 进程，以使用我们训练的模型来控制模拟的 Donkey（汽车）。

要开始使用这个环境，请查看以下资源：
- [DonkeyCar 模拟器文档](https://docs.donkeycar.com/guide/deep_learning/simulator/)
- [学习平稳驾驶（Antonin Raffin 的教程）第1部分](https://www.youtube.com/watch?v=ngK33h00iBE)
- [学习平稳驾驶（Antonin Raffin 的教程）第2部分](https://www.youtube.com/watch?v=DUqssFvcSOY)
- [学习平稳驾驶（Antonin Raffin 的教程）第3部分](https://www.youtube.com/watch?v=v8j2bpcE4Rg)

- 预训练的智能体：
  - https://huggingface.co/araffin/tqc-donkey-mountain-track-v0
  - https://huggingface.co/araffin/tqc-donkey-avc-sparkfun-v0
  - https://huggingface.co/araffin/tqc-donkey-minimonaco-track-v0


## 星际争霸 II

<img src="https://huggingface.co/datasets/huggingface-deep-rl-course/course-images/resolve/main/en/unit12/alphastar.jpg" alt="Alphastar"/>

《星际争霸II》是一款著名的实时战略游戏。DeepMind 利用这款游戏进行他们的深度强化学习研究，其中包括 [Alphastar](https://www.deepmind.com/blog/alphastar-mastering-the-real-time-strategy-game-starcraft-ii)。

要开始使用这个环境，请查看以下资源：
- [星际争霸 Gym](http://starcraftgym.com/)
- [AI学习玩《星际争霸2》（强化学习）教程](https://www.youtube.com/watch?v=q59wap1ELQ4)
